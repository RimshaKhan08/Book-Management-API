# Book-Management-API


Developed a REST API to manage books using FastAPI and Python. Implemented full CRUD operations (Create, Read, Update, Delete) and query/filtering functionality by author and category.

Key Features:
Retrieve all books or a specific book by title
Filter books by category or author
Add new books via POST request
Update existing books via PUT request
Delete books via DELETE request
Implemented case-insensitive search for better usability

Skills & Tools:
Python, FastAPI, REST API, CRUD operations, Backend Development, API Design
